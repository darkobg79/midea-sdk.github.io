MideaSDK-iOS 常见问题汇总

2017年12月20日

一些常见问题：

1、哪里获取最新的 MideaSDK ？

开放平台 和 GitHub 均支持下载，其中使用 CocoaPod 来管理 MideaSDK 的开放者，对应的 MideaSDK 是 GitHub 上的。

Issues

2、为什么MideaSDK是以.framework为后缀，但是实际却是是静态库，而非动态库？

因为 MideaSDK 最低兼容iOS7，在iOS8之前，iOS平台不支持使用动态 framework。一旦 MideaSDK 将支持版本从iOS8开始，我们将处理这个问题。

3、开放过程中遇到一些问题，如何反馈？

可以先在这里 GitHub-Issues 上查询是否有类似的问题，如果没有找到，欢迎将问题提交在 GitHub-Issues 上，或者邮件 pan.wang@midea.com。
